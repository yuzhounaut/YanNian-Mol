Lifespan Predictor Documentation
==================================

Welcome to the Lifespan Predictor documentation! This package provides a modular, efficient machine learning pipeline for predicting compound effects on *C. elegans* lifespan using multi-modal deep learning.

Features
--------

* **Multi-Modal Architecture**: Combines Graph Neural Networks (AttentiveFP), CNNs, and DNNs
* **Efficient Caching**: Disk-based caching for molecular features and fingerprints
* **Parallel Processing**: Multi-core support for featurization
* **Flexible Configuration**: YAML-based configuration with validation
* **Comprehensive Metrics**: Support for classification and regression tasks
* **Training Callbacks**: Early stopping, checkpointing, and learning rate scheduling
